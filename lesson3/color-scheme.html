Color Scheme

The color palette for the site is:

The background color for the site is EAEBED. All text in the navagation, header, footer that is on the main and secondary colors will be EAEBED as well.

The main color on the site will be 006989. This will be the backgroud color for the footer elements.

The secondary color will be 6B4B3E. This color will be used as a background for any objects within the main body of the site.

The accent colors are FE654F and F49D37. These will be used in coordination with the main and secondary colors respectively.
